# Arduino Dino Run

This repo is forked from [kiedtl/DinoRun](https://github.com/kiedtl/DinoRun) .

# Getting Started !

```sh
$ git clone ...
$ cd DinoRun
```

# Installation !
`Note : You must have nodeJS installed in your computer.`

```sh
$ npm install
```
With Permission 
```sh
$ sudo npm install
```
